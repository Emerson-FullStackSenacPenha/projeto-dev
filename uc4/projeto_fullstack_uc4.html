Configurando o Ambiente de Desenvolvimento


XAMPP

  • Pacote que contém PHP, MySQL, Apache e Perl
  • Apache: simula um servidor web no computador
  • MySQL: banco de dados para armazenar informações

Preparando o Ambiente

  • Instalar o XAMPP e marcar todos os softwares durante a instalação
  • Criar a pasta do projeto:
    C:\xampp\htdocs\introducao_php
  • Iniciar o Apache no XAMPP
  • Instalar extensão "Intelephense" no VS Code para entender PHP
  • Acessar pelo navegador:
    http://localhost/introducao_php/ola_mundo.php

JavaScript x PHP

  • JavaScript: roda no navegador, código visível
  • PHP: roda no servidor, mais seguro e invisível ao usuário

JSON – Organização de Dados


O que é JSON?

  • JSON (JavaScript Object Notation) é uma forma de organizar dados
  • Permite que sistemas diferentes troquem informações de forma simples
  • É leve, fácil de ler e entender tanto para humanos quanto para máquinas

Estrutura do JSON

  • Baseado em chave-valor:
    • {
  "nome": "João",
  "idade": 25
}
  • Valores podem ser números, textos, arrays ou outros objetos

Exemplo de Array em JSON

{
  "frutas": ["maçã", "banana", "laranja"]
}

Analogia

Imagine JSON como uma **caixa de organizador**:
  • Cada chave é um compartimento com um rótulo
  • O valor é o objeto guardado dentro do compartimento
  • Facilita encontrar e trocar informações rapidamente

Uso Prático

  • APIs usam JSON para enviar dados entre sistemas
  • JavaScript e PHP conseguem ler e escrever JSON facilmente
  • Permite salvar dados temporários ou enviar informações para um servidor

Conceitos Fundamentais de Programação


Componentes de uma Linguagem

  • Entrada: organizar e entender os dados que chegam
  • Processamento: aplicar lógica e funções para resolver problemas
  • Abstração: transformar problemas reais em elementos que o computador entende
  • Saída: mostrar o resultado ao usuário, por exemplo, na tela

Analogia

Imagine a programação como uma fábrica:
  • Entrada = Matéria-prima (dados)
  • Processamento = Máquinas que transformam os dados (funções e lógica)
  • Saída = Produto final entregue ao cliente (resultado na tela)

Variáveis no PHP


Tipos de Variáveis

  • Simples: guarda um único valor (ex.: número ou texto)
  • Compostas: arrays que guardam múltiplos valores, podem ser indexados ou associativos

Características do PHP

  • Alto nível: fácil de entender, próximo da linguagem humana
  • Dínamico: não precisa declarar tipo da variável
  • Interpretado: roda sem precisar compilar
  • Tipagem fraca: tipo da variável pode mudar facilmente
  • Multi-paradigma: permite escrever de várias formas
  • Case sensitive: diferencia maiúsculas de minúsculas

Fluxo de Programação


Resumo

  • Entrada: abstrair o problema e organizar dados
  • Processamento: aplicar lógica e resolver o problema
  • Saída: mostrar resultado ao usuário

Exemplo Prático

Entrada: número do usuário
Processamento: calcular dobro
Saída: mostrar resultado
