Trabalhando com imagens

Obs.:

Abra o código fonte do seu navegador para entender as tags. (Chrome Ctrl+U)

Para exibir imagens em uma página web, usamos a tag < img >. Ela é auto-fechada e precisa de, no mínimo, dois atributos: src (fonte da imagem) e alt (texto alternativo).

Formatos mais comuns

  • JPEG/JPG: Muito usado para fotos em geral
  • PNG: Imagens com exigência maior de qualidade, permite uso de transparência.Pode ser mais pesado.
  • GIF: Usado para desenhos/ilustrações, com poucas cores (até 256 cores). Suporte báscio à transparência e permite animações.
  • WEBP: Formato moderno, com ótima compressão, suporta transparência e animação.
  • SVG: Formato de imagem vetorial, muito usado em desenhos e logotipos, que pode ser exportado a partir de programas como CorelDRAW/Ilustrator. Um formato de imagem vetorial baseado em código XML, usado principalmente para ícones, logos e gráficos que precisam manter qualidade em qualquer tamanho.

Imagem/Ilustração COM legenda

A tag < figure > é usada para agrupar uma imagem (ou outro conteúdo visual) com uma legenda descritiva, usando a tag < figcaption >. Isso dá mais significado semântico ao conteúdo.
