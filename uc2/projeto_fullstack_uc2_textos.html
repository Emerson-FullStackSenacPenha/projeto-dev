Elementos textuais


Neste conteúdo, aprendemos os conceitos iniciais do desenvolvimento web voltado ao Front-End, que é a parte visual dos sites, feita para interagir com os usuários.

Títulos e subtítulos


As tags < h1 > até < h6 > são usadas para definir títulos e subtítulos em uma página HTML. Elas indicam a hierarquia de importância dos conteúdos. < h1 > é o título mais importante, geralmente usado para o título principal da página. < h2 > a < h6 > são usados como subtítulos, com importância decrescente, essas tags ajudam na organização do conteúdo, melhoram a acessibilidade e são muito importantes para o SEO (otimização para mecanismos de busca).

Estilos de texto


Negrito: Destaca um texto com ênfase forte. < b >


Itálico: Dá ênfase ou indica algo especial como nomes ou termos estrangeiros. < i >


Sublinhado: Sublinha o texto, geralmente usado para chamar atenção (com moderação). < u >


Riscado: Indica algo removido ou inválido. < s >

Abreviações


A tag < abbr > é usada para marcar abreviações ou siglas, permitindo que o navegador exiba o significado completo ao passar o mouse sobre o texto.

Conteúdo Expansível


A tag < details > é usada para esconder e revelar conteúdo quando o usuário clica em um resumo ou título. É muito útil para criar abas, perguntas frequentes (FAQ), ou informações extras sem deixar a página poluída.
