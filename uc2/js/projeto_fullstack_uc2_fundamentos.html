Introdução de JavaScript


Resumo da primeira aula de Js com o professor Tiago

Introdução e Ambiente


Sequência ideal para aprender desenvolvimento web:

  • HTML
  • CSS
  • JavaScript (por último, para adicionar interatividade)

Inclusão do JavaScript no HTML:


Inserir o <script src="arquivo.js"></script> no final do <body>.

Se precisar colocar o script no <head>, usar o atributo defer para garantir que o HTML carregue antes do JS executar, evitando bugs.

Operações Aritméticas


Use parênteses para definir prioridade nas expressões.


Exemplos:


let resultado = (2 + 3) * 4; // resultado = 20

let resultado = 2 + 3 * 4; // resultado = 14

Variáveis e Constantes


Variável (dados que mudam): declarada com let.

Constante (dados fixos): declarada com const.


Exemplo:

let nome = "João"; // variável dinâmica

const pi = 3.14; // valor fixo

Concatenar Strings


Forma antiga (Concatenação tradicional):

Usar aspas "" e sinal de + para juntar textos e variáveis.


console.log("Olá " + nome + ", tudo bem?");

Forma moderna (Template Literals):

Usar crase e ${} para inserir variáveis e expressões, mais legível e com suporte a quebras de linha.


console.log(`Olá ${nome}, tudo bem?`);

Quebra de linha em strings


Pode concatenar com +=:

let texto = "Olá, sou fulano ";
texto += "e sou tal coisa.";

Template literals já permitem quebras de linha naturalmente:

let texto = `Linha 1
Linha 2`;

Usando aspas dentro de texto


Errado

// console.log("Gosto "muito" de paçoca");  // ERRADO

Correto

console.log('Gosto "muito" de paçoca'); // aspas simples no comando, duplas no texto
console.log("Gosto 'muito' de paçoca"); // aspas duplas no comando, simples no texto
console.log("Gosto \"muito\" de paçoca"); // escapando aspas com \

Modo Estrito (Strict Mode)


Coloque no início do arquivo .js para ativar o modo estrito, que ajuda a evitar erros comuns, como usar variáveis não declaradas:


'use strict';
teste = "Senac"; // erro, variável não declarada
console.log(teste);

Tipos de Dados e Verificação


Para saber o tipo de uma variável, usar:


console.log(typeof nome); "string"
console.log(typeof pi);   "number"
