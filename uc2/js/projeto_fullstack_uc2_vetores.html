Trabalhando com Vetores (Arrays)


O que são vetores:
Vetores (arrays) guardam vários dados em uma única variável.


Índices:
Cada elemento tem uma posição chamada índice, começando em 0.


Exemplo de Array e Strings

let vocal = ["Ozzy", "Dio", "Lemmy", "David"];
let msg = "O vocalista ";

console.log(msg + vocal[0] + " é famoso.");
// Saída: O vocalista Ozzy é famoso.

console.log(msg + vocal[3] + " toca Pink Floyd.");
// Saída: O vocalista David toca Pink Floyd

console.log(`Último: ${vocal[3]}`);
// Saída: Último: David

Explicação:
- Arrays armazenam múltiplos valores em uma variável.
- Strings podem ser concatenadas usando + ou template literals ${}.
- Cada elemento do array é acessado pelo índice [].


Resumo:
Vetores e strings juntos permitem criar mensagens dinâmicas de forma simples.
Use índices para acessar os dados e concatenar ou interpolar com template literals.

12. Manipulação do DOM, Arrays e Objetos


Arrays heterogêneos:
No JS, um array pode conter dados de tipos diferentes.


Arrays de 1 dimensão:
Já aprendemos sobre arrays simples (1D), que armazenam uma lista de elementos sequenciais.


Matrizes (arrays 2D):
São arrays de duas dimensões, como uma planilha Excel, acessados por linha e coluna (índices começam em 0).


Arquivos JS:
Crie um arquivo externo e linke no HTML:
<script src="js/04-objetos.js"></script>
Mesmo sem criar ainda, já nomeie o arquivo no editor, segure CTRL + clique e selecione "Criar Arquivo".


Objetos:
Objetos representam características de algo, por exemplo, um cliente.
- () Parênteses → operações e funções
- [] Colchetes → arrays
- {} Chaves → objetos, funções, condicionais, loops, template strings


Exemplos de Objetos e Arrays

let pessoa = {nome: "Tiago", idade: 30};
console.log(`O ${pessoa.nome} tem ${pessoa.idade} anos.`);
// Saída: O Tiago tem 30 anos.

let livros = [
  {titulo: "Livro 1"},
  {titulo: "Livro 2"},
  {titulo: "Livro 3"}
];
console.log(livros[2].titulo);
// Saída: Livro 3

console.log("Exemplo 1\nExemplo 2");
// Saída: Exemplo 1
//         Exemplo 2

Explicação:
- Use o ponto . para acessar propriedades de objetos.
- Use colchetes [] para acessar elementos de arrays.
- Template literals ${} permitem combinar variáveis e texto.
- \n pula linha no console.


Resumo:
Arrays e objetos permitem organizar dados de forma estruturada.
Combine com template literals e console.log para exibir informações de forma clara.

13. Caracteres, Condicionais e Operadores Lógicos


Caracteres principais:
() → parênteses para operações
[] → colchetes para arrays/vetores
{} → chaves para objetos, funções, condicionais, loops e template strings


Condicionais:
- if → SE
- else → SENÃO
- Condicional comum: if
- Condicional composta: if else
- Para uma linha, pode usar operador ternário condição ? valor1 : valor2


Exemplo de Condicional

let idade = 20;
let temCarta = true;

if (idade >= 18 && temCarta) {
    console.log("Pode dirigir");
} else {
    console.log("Não pode dirigir");
}
// Saída: Pode dirigir

Operadores Lógicos:
&& → E (AND, todos devem ser verdadeiros)
|| → OU (OR, apenas um precisa ser verdadeiro)
! → NÂO/negação (NOT, inverte o valor)


Comparadores:
== → compara valor
=== → compara valor e tipo (mais seguro para evitar erros de tipo)


Prioridade de execução:
1. !
2. &&
3. ||


Resumo:
Use caracteres corretos para cada estrutura.
Condicionais controlam o fluxo do programa.
Operadores lógicos e comparadores ajudam a criar decisões complexas de forma clara.

14. Loops e Estruturas de Repetição


O que são loops:
Loops executam ações repetidamente com base em uma condição lógica.


Tipos de loops:
- for → Para arrays ou contadores
- for of → Exclusivo JS, percorre valores de arrays
- for in → Exclusivo JS, percorre propriedades de objetos
- while → Enquanto a condição for verdadeira


Variáveis de controle:
- Normalmente chamadas de i, j, k ou qualquer letra
- Incrementos: i++ → soma 1
- Incrementos personalizados: i += 5 → soma 5


Exemplo: Loop For

const bandas = ["Nirvana", "Pearl Jam", "Metallica"];

for(let i = 0; i < bandas.length; i++) {
    console.log(bandas[i]);
}
// Saída: Nirvana
//         Pearl Jam
//         Metallica

Exemplo: Loop For Of

for(let banda of bandas) {
    console.log(banda);
}
// Saída igual ao anterior

Exemplo: Loop For In (Objetos)

const pessoa = {nome: "Tiago", idade: 30};

for(let prop in pessoa) {
    console.log(prop + ": " + pessoa[prop]);
}
// Saída: nome: Tiago
//         idade: 30

Explicação:
- Use loops para percorrer arrays ou objetos automaticamente.
- length define quantas vezes um loop deve rodar com base no tamanho do array.
- for of percorre valores diretamente, for in percorre propriedades.


Resumo:
Loops permitem automatizar tarefas repetitivas.
Escolha o tipo adequado: for, for of, for in ou while.

15. Arrays, Objetos e Loops Explicados de Forma Simples


Arrays (Vetores)
Um array é como uma caixinha com vários brinquedos dentro.
Cada brinquedo tem uma posição, começando do 0.


let frutas = ["maçã", "banana", "uva"];
console.log(frutas[0]); // maçã
console.log(frutas[2]); // uva

Objetos
Um objeto é como uma caixinha com etiquetas.
Cada etiqueta tem um nome e um valor.


let pessoa = {nome: "Ana", idade: 5};
console.log(pessoa.nome); // Ana
console.log(pessoa.idade); // 5

Arrays dentro de Objetos
Um objeto pode ter uma caixinha de brinquedos dentro.


let caixa = {brinquedos: ["bola", "boneca", "carrinho"]};
console.log(caixa.brinquedos[1]); // boneca

Objetos dentro de Arrays
Um array pode ter várias caixinhas de brinquedos.


let caixaArray = [
  {nome: "Ana", idade: 5},
  {nome: "Leo", idade: 6}
];
console.log(caixaArray[1].nome); // Leo

Loops (Repetições)
Loops repetem coisas várias vezes, como contar brinquedos.


While

let i = 0;
while(i < 3) {
  console.log("Brinquedo " + i);
  i++;
}
// Saída: Brinquedo 0, Brinquedo 1, Brinquedo 2

For

for(let i = 0; i < 3; i++) {
  console.log("Brinquedo " + i);
}

For Of (para arrays)

let frutas = ["maçã", "banana", "uva"];
for(let fruta of frutas) {
  console.log(fruta);
}

For In (para objetos)

let pessoa = {nome: "Ana", idade: 5};
for(let prop in pessoa) {
  console.log(prop + ": " + pessoa[prop]);
}

Resumo:
- Arrays são caixinhas com brinquedos.
- Objetos são caixinhas com etiquetas e valores.
- Loops repetem coisas várias vezes.
- Você pode misturar arrays e objetos para organizar tudo de forma divertida!
