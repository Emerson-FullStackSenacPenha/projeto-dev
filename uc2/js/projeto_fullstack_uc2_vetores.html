Introdução Arrays / Vetores

Explicação:
Arrays são estruturas que armazenam múltiplos valores em uma única variável.
Cada valor dentro do array é chamado de elemento e possui uma posição chamada índice.
O índice sempre começa em 0, ou seja, o primeiro elemento está na posição 0.
Arrays são muito usados quando precisamos organizar dados relacionados de forma sequencial.

Uma forma fácil de entender arrays é pensar em um armário com várias gavetas.
Cada gaveta tem uma posição (índice) e contém algum item (elemento).
Para pegar algo específico, basta indicar a gaveta correspondente.

Como declarar um Array

Explicação:
Um array é declarado usando colchetes [].
Cada elemento é separado por vírgula.

let armario = ["camisa", "calça", "meias", "chapeu", "sapatos"];

Acessando elementos do Array

Explicação:
Para acessar um elemento, usamos o nome do array seguido do índice entre colchetes.
O índice indica a posição da gaveta que queremos abrir.

console.log(armario[0]); // "camisa"
console.log(armario[4]); // "sapatos"

Modificando elementos

Explicação:
Podemos alterar o conteúdo de uma posição específica do array.
Basta acessar o índice e atribuir um novo valor.

armario[0] = "jaqueta";
console.log(armario[0]); // "jaqueta"

Propriedades importantes

  • length: indica quantos elementos o array possui.
  • push(): adiciona um elemento no final do array.
  • pop(): remove o último elemento do array.
  • shift(): remove o primeiro elemento do array.
  • unshift(): adiciona um elemento no início do array.

Resumo dos Arrays

Resumo:
Arrays são como armários com gavetas.
Cada gaveta tem uma posição (índice) e contém um elemento.
Podemos acessar, modificar ou adicionar elementos indicando a posição.
Arrays facilitam organizar e manipular conjuntos de dados relacionados.
Propriedades como length e métodos como push/pop ajudam a gerenciar o conteúdo do array.
