O que são Loops em JavaScript


Explicação:
Loops são como "rotinas automáticas" que repetem uma tarefa várias vezes até que uma condição seja atendida.
Pense em um loop como um porteiro que verifica uma fila: enquanto houver pessoas, ele continua chamando uma por uma.
Em programação, usamos loops para percorrer dados, executar cálculos ou automatizar tarefas repetitivas.

Resumo:
Loops ajudam a economizar código e executar ações repetidas de forma organizada.
São fundamentais para lidar com listas, cálculos e estruturas de dados.

Loop While


Explicação:
O "while" significa "enquanto". Imagine que você está enchendo uma caixa de bolas:
enquanto houver espaço, continue colocando mais bolas.
O código repete a ação enquanto a condição for verdadeira.

Exemplo:


let numero = 2;
while (numero <= 10) {

Concatenando strings com o resultado
console.log("Número atual: " + numero + " - executando ação...");

Usando template string com resultado
console.log(`O dobro de ${numero} é ${numero * 2} - ação executada!`);

  numero = numero + 2;
}

// Saída no console:
// Número atual: 2 - executando ação...
// O dobro de 2 é 4 - ação executada!
// Número atual: 4 - executando ação...
// O dobro de 4 é 8 - ação executada!
// ... até 10

Resumo:
"while" executa uma tarefa repetidamente até que a condição se torne falsa.
Aqui mostramos como exibir valores usando concatenação e template string, simulando a saída do console.

Loop For


Explicação:
O "for" é como uma agenda: você já sabe quantas tarefas precisa fazer.
Ele é útil quando sabemos o número exato de repetições, como contar de 1 a 10.

Exemplo:


for (let i = 1; i <= 5; i++) {

Concatenando strings com resultado
console.log("Valor atual: " + i + " - item processado.");

Usando template string com resultado
console.log(`O triplo de ${i} é ${i * 3} - processado com sucesso!`);
}

// Saída no console:
// Valor atual: 1 - item processado.
// O triplo de 1 é 3 - processado com sucesso!
// Valor atual: 2 - item processado.
// O triplo de 2 é 6 - processado com sucesso!
// ... até 5

Resumo:
"for" é ótimo para quando você conhece o ponto de início, fim e o passo de repetição.
Também permite formatar a saída usando concatenação e template string, mostrando os resultados.

Loop For...In


Explicação:
"for...in" é como olhar para as gavetas de um armário: você não está interessado no que há dentro ainda, mas quer saber o nome de cada gaveta.
Ele percorre as propriedades (chaves) de um objeto.

Exemplo:


const aluno = { nome: "Ana", idade: 20, curso: "ADS" };
for (let chave in aluno) {

Concatenando strings com resultado
console.log("Propriedade: " + chave + " - encontrada no objeto.");

Usando template string com resultado
console.log(`Chave ${chave} tem o valor ${aluno[chave]}-verificado!`);
}

// Saída no console:
// Propriedade: nome - encontrada no objeto.
// A chave nome tem o valor Ana - verificado!
// Propriedade: idade - encontrada no objeto.
// A chave idade tem o valor 20 - verificado!
// Propriedade: curso - encontrada no objeto.
// Chave curso tem o valor ADS - verificado!

Resumo:
"for...in" percorre as chaves (propriedades) de um objeto e permite mostrar suas informações no console.

Loop For...Of


Explicação:
"for...of" é como abrir cada gaveta e olhar dentro: você vê os itens armazenados.
Ele percorre os valores de arrays, strings ou outras coleções iteráveis.

Exemplo:


const frutas = ["maçã", "banana", "uva"];
for (let fruta of frutas) {

  // Concatenando strings com resultado
  console.log("Fruta: " + fruta + " - adicionada ao carrinho.");

  // Usando template string com resultado
  console.log(`Fruta atual: ${fruta} - compra registrada!`);
}

// Saída no console:
// Fruta: maçã - adicionada ao carrinho.
// Fruta atual: maçã - compra registrada!
// Fruta: banana - adicionada ao carrinho.
// Fruta atual: banana - compra registrada!
// Fruta: uva - adicionada ao carrinho.
// Fruta atual: uva - compra registrada!

Resumo:
"for...of" percorre diretamente os valores de arrays e outros iteráveis, permitindo mostrar mensagens realistas no console.

Exercício: Objetos dentro de um Array


Explicação:
Agora vamos trabalhar com um array que contém vários objetos. Cada objeto representa um cliente com nome e ID.
O objetivo é percorrer todos os clientes usando diferentes tipos de loops e imprimir as informações no formato:
Cliente: "Nome" ID: "Número"


Array de Clientes


const clientes = [
  { nome: "Ana", id: 101 },
  { nome: "Bruno", id: 202 },
  { nome: "Carla", id: 303 }
];

Usando While


let i = 0;
while (i < clientes.length) {

Concatenando strings com resultado
console.log("Cliente: "+clientes[i].nome + " ID: "+clientes[i].id);

Usando template string com resultado
console.log(`Cliente: ${clientes[i].nome} ID: ${clientes[i].id}`);

  i++;
}

// Saída no console:
// Cliente: Ana ID: 101
// Cliente: Bruno ID: 202
// Cliente: Carla ID: 303

Usando For


for (let i = 0; i < clientes.length; i++) {

Concatenando strings com resultado
console.log("Cliente: "+clientes[i].nome+" ID: "+clientes[i].id);

Usando template string com resultado
console.log(`Cliente: ${clientes[i].nome} ID: ${clientes[i].id}`);
}

// Saída no console:
// Cliente: Ana ID: 101
// Cliente: Bruno ID: 202
// Cliente: Carla ID: 303

Usando For...In


for (let ind in clientes) {

Concatenando strings com resultado
console.log("Cliente: "+clientes[ind].nome+" ID: "+clientes[ind].id);

Usando template string com resultado
console.log(`Cliente: ${clientes[ind].nome} ID: ${clientes[ind].id}`);
}

// Saída no console:
// Cliente: Ana ID: 101
// Cliente: Bruno ID: 202
// Cliente: Carla ID: 303

Usando For...Of


for (let cliente of clientes) {

Concatenando strings com resultado
console.log("Cliente: " + cliente.nome + " ID: " + cliente.id);

Usando template string com resultado
console.log(`Cliente: ${cliente.nome} ID: ${cliente.id}`);
}

// Saída no console:
// Cliente: Ana ID: 101
// Cliente: Bruno ID: 202
// Cliente: Carla ID: 303

Resumo:
Percorremos um array de objetos com os loops While, For, For...In e For...Of.
Cada exemplo mostra como acessar e exibir os dados de cada cliente de duas formas: concatenação de strings e template string.
