Refatoração

Explicação:
Refatorar significa melhorar ou reescrever um código já existente sem mudar sua função.
É como arrumar seu quarto: você não troca os móveis de lugar, mas deixa tudo mais limpo e organizado.

Exemplo prático

// Antes
let x = 1; let y = 2; let z = x + y;

// Depois da refatoração
let x = 1;
let y = 2;
let soma = x + y;

Resumo:
Refatorar = deixar o código mais claro e organizado.
Não muda o resultado final, apenas melhora a forma de escrever.

Métodos de Classe: add, remove e toggle

Explicação:
Esses métodos são usados para adicionar, remover ou alternar classes em elementos HTML.
Imagine que as classes são "etiquetas" que grudamos em um objeto. Podemos ligar, desligar ou alternar essas etiquetas.

  • add: Liga (adiciona uma classe)
  • remove: Desliga (remove uma classe)
  • toggle: Alterna (liga ou desliga, dependendo do estado atual)

Exemplo prático

const caixa = document.querySelector("#caixa");

// Liga
caixa.classList.add("ativo");

// Desliga
caixa.classList.remove("ativo");

// Alterna
caixa.classList.toggle("ativo");

Resumo:
add = ligar uma classe.
remove = desligar uma classe.
toggle = alternar entre ligar e desligar.

Dialog (Janela de Diálogo)

Explicação:
O elemento <dialog> é usado para criar janelas modais.
É como uma caixa de recado que aparece na frente da página, pedindo atenção do usuário.

Exemplo prático

<dialog id="janela">
  <p>Bem-vindo!</p>
</dialog>

<button id="abrir">Abrir</button>

<script>
const janela = document.querySelector("#janela");
const abrir = document.querySelector("#abrir");

abrir.addEventListener("click", () => {
  janela.showModal();
});
</script>

Resumo:
Dialog = janela de diálogo.
Usada para mostrar informações ou pedir uma ação ao usuário.

Caixa de Senha

Explicação:
O campo de senha esconde os caracteres digitados para garantir privacidade.
Podemos adicionar um botão para mostrar ou esconder a senha quando necessário.

Exemplo prático

<input type="password" id="senha">
<button id="mostrar">Mostrar senha</button>

<script>
const senha = document.querySelector("#senha");
const botao = document.querySelector("#mostrar");

botao.addEventListener("click", () => {
  if (senha.type === "password") {
    senha.type = "text";
  } else {
    senha.type = "password";
  }
});
</script>

Resumo:
Caixa de senha esconde os caracteres.
Podemos alternar entre "password" e "text" para mostrar ou ocultar a senha.

Caixa de Busca

Explicação:
O campo de busca ajuda o usuário a encontrar algo dentro do site.
É como uma lupa em um livro: você digita e ele procura pelo conteúdo desejado.

Exemplo prático

<input type="search" id="busca" 
 placeholder="Buscar...">

<script>
const busca = document.querySelector("#busca");

busca.addEventListener("input", () => {
  console.log("Procurando por:", busca.value);
});
</script>

Resumo:
Caixa de busca permite digitar termos para localizar informações.
Usada em sites, lojas virtuais e aplicativos.
