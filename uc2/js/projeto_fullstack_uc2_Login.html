O que é o DOM (Modelo de objetos do documento)


Explicação:
O DOM é como uma "árvore genealógica" dos elementos de uma página HTML.
Cada tag (como <h1>, <p>, <div>) é tratada como um objeto dentro dessa árvore.
Isso permite que o JavaScript consiga acessar e manipular esses elementos, alterando conteúdo, estilos e até criando novos nós.


Estrutura do DOM


  • Raiz: <html>
  • Filhos diretos: <head> e <body>
  • Netos: <title>, <h1>, <p>, etc.

Exemplo prático


// Procurando elementos no DOM
const exemplo01 = document.querySelector("#exemplo1");
const mensagem01 = document.querySelector("#mensagem01");
const pagina = document.querySelector("body");

console.log(exemplo01);
console.log(mensagem01);
console.log(pagina);

Resumo:
O DOM transforma o HTML em uma árvore de objetos manipuláveis.
Com ele, podemos selecionar, modificar e interagir com qualquer parte da página.
É a ponte entre HTML (estrutura) e JavaScript (funcionalidade).

Seletores no DOM


Explicação:
Para manipular o DOM, precisamos "apontar" para o elemento correto.
Os seletores são ferramentas que nos ajudam a encontrar esses elementos.


Tipos de Seletores


  • querySelector() → seleciona o primeiro elemento que corresponde ao seletor.
  • querySelectorAll() → seleciona vários elementos.
  • getElementById() → seleciona um elemento específico pelo ID.

Exemplo prático


// Selecionando diferentes formas
const item = document.querySelector("p"); // primeiro parágrafo
const lista = document.querySelectorAll("li"); // todos os itens de lista
const titulo = document.getElementById("titulo-principal"); // por ID

Resumo:
Os seletores permitem "encontrar" os elementos que queremos manipular.
Podemos escolher por tag, classe, ID ou até combinações de seletores.

Eventos em JavaScript


Explicação:
Um evento é como um "sinal" de que algo aconteceu na página.
Pode ser um clique, uma tecla pressionada, a rolagem ou o carregamento da página.
O JavaScript permite "ouvir" esses eventos e reagir a eles.


Exemplos comuns de eventos


  • Clique do mouse
  • Pressionar teclas
  • Toque na tela
  • Rolagem da página
  • Carregamento da página

Função para monitorar eventos


// Criando um evento de clique
const exemplo01 = document.querySelector("#exemplo1");

exemplo01.addEventListener("click", function() {
  alert("Você clicou no exemplo!");
});

Resumo:
Eventos são acontecimentos que disparam ações no código.
O addEventListener é a forma mais comum de "ouvir" e reagir a esses eventos.
Isso torna a página interativa e dinâmica.

Exemplo: Criando um Modo Noturno


Explicação:
Podemos usar eventos para criar recursos práticos, como alternar entre modo claro e escuro em um site.
Usaremos o método toggle() para alternar uma classe no body sempre que um botão for clicado.


Exemplo prático


// Selecionando botão e body
const botao = document.querySelector("#btn-noturno");
const body = document.querySelector("body");

// Criando evento de clique
botao.addEventListener("click", function() {
  body.classList.toggle("modo-escuro");
});

Resumo:
O método toggle() alterna uma classe: se ela existir, remove; se não existir, adiciona.
Esse é um jeito simples de criar o "modo noturno" em qualquer site.
