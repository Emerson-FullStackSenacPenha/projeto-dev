Estruturas Combinadas em JavaScript

Explicação:
Em JavaScript, podemos combinar Arrays e Objetos para organizar dados mais complexos.
Duas estruturas comuns são: - Objeto dentro de Array: várias gavetas (objetos) dentro de um armário (array). - Array dentro de Objeto: uma gaveta (objeto) que contém vários itens (array).

Essas combinações ajudam a representar situações do mundo real, como armários com várias gavetas e organizadores internos.

Objeto dentro de Array

Explicação:
Um array pode conter múltiplos objetos, cada um representando um armário.
Cada objeto tem suas próprias gavetas (propriedades) e conteúdos (valores).
Para acessar, usamos o índice do array e depois a chave do objeto.


let armarios = [
  { camisa: "verde", calca: "amarela" },
  { camisa: "preta", calca: "branca" }
];

console.log(armarios[1].camisa); // "preta"

Array dentro de Objeto

Explicação:
Um objeto pode ter uma propriedade que é um array.
Cada propriedade funciona como uma gaveta do armário, e o array dentro dela representa os itens dessa gaveta.
Para acessar, usamos o nome do objeto, a chave e o índice do array.


let armario = {
  gavetaDeMeias: ["azul", "vermelha", "preta"]
};

console.log(armario.gavetaDeMeias[2]); // "preta"

Como acessar e modificar

Explicação:
Podemos combinar as estruturas para acessar ou modificar elementos específicos.
Exemplo: alterar a primeira meia da gaveta do armário dentro de um array.


let quarto = [
  { gavetaDeMeias: ["azul", "verde"] },
  { gavetaDeMeias: ["preta", "branca"] }
];

// Alterar a primeira meia do primeiro armário
quarto[0].gavetaDeMeias[0] = "rosa";

console.log(quarto[0].gavetaDeMeias[0]); // "rosa"

Resumo das Estruturas Combinadas

Resumo:
Objetos dentro de Arrays representam múltiplos armários com gavetas próprias.
Arrays dentro de Objetos representam gavetas com vários itens.
Podemos acessar, modificar e percorrer essas estruturas combinadas usando índices e chaves.
Essas estruturas ajudam a modelar dados complexos de forma clara e organizada, como armários com gavetas e organizadores internos.
