Resumo: Funções


Explicação:
Funções são blocos de código que executam tarefas específicas.
Podem receber dados (parâmetros) e devolver resultados (retorno).
Ajudam a organizar, reutilizar e simplificar o código.


Quando usar:


  • Evitar repetir código.
  • Separar tarefas em blocos claros.
  • Executar operações sob demanda.

Resumo:
- Organizam e reaproveitam código.
- Facilitam manutenção e leitura.
- Permitem criar programas mais eficientes.


Função Anônima


Explicação:
Uma função anônima é aquela que não possui nome próprio. Ela é atribuída a uma variável ou constante, que passa a ser a referência para chamá-la.
O nome da constante ou variável é o que usamos para invocar essa função.
Não pode ser executada antes de ser declarada, pois o JavaScript só conhece sua existência após a linha de atribuição.
É necessário usar o comando return de forma explícita caso queira devolver algum valor.


Exemplo:



const exemplo1 = function() {
    console.log("Olá, função anônima!");
};

exemplo1(); // chamada

Resumo:
- Função sem nome, atribuída a uma variável ou constante.
- Só pode ser chamada após ser criada.
- Precisa de return explícito para devolver valores.


Função Nomeada/Declarada


Explicação:
Uma função nomeada é criada com a palavra-chave function e um nome. Pode ser chamada antes ou depois da sua declaração devido ao Hoisting (elevação), recurso do JavaScript que “move” a função para o topo do código em tempo de execução.


Exemplo:



function exemplo2() {
    console.log("Exemplo de função nomeada");
}

exemplo2(); // chamada

Resumo:
- Possui nome próprio.
- Pode ser executada antes de sua declaração (Hoisting).
- Facilita leitura e manutenção do código.


Função Arrow (Função Flecha)


Explicação:
A Arrow Function é uma forma mais curta de escrever funções em JavaScript. Ela é sempre atribuída a uma variável ou constante. Não sofre hoisting, por isso precisa ser declarada antes de ser chamada.
Útil para simplificar códigos curtos, especialmente em callbacks.


Exemplo:



const exemplo3 = () => {
    console.log("Sintaxe Arrow Function!");
};

exemplo3();

Resumo:
- Sintaxe curta e moderna.
- Não sofre elevação (Hoisting).
- Ideal para funções rápidas e reutilizáveis.


Função com Parâmetro


Explicação:
Funções podem receber dados externos chamados parâmetros. Eles permitem tornar o código mais flexível.
É possível definir valores padrão para parâmetros, evitando erros quando não são informados.


Exemplo:



function saudacao(nome = "Visitante") {
    console.log("Olá, " + nome);
}

saudacao("Emerson");
saudacao();

Resumo:
- Parâmetros tornam a função adaptável.
- Valores padrão evitam falhas sem argumento.
- Permitem personalizar resultados.


Função com Parâmetros e Retorno


Explicação:
Além de receber valores, funções podem devolver resultados com return. Esse retorno pode ser armazenado em variáveis para uso posterior.


Exemplo:



function multiplicar(valor1, valor2) {
    return valor1 * valor2;
}

let resultado1 = multiplicar(10, 5);
console.log("Resultado 1: " + resultado1);

Resumo:
- return devolve o resultado.
- Retornos podem ser usados em outras operações.
- Aumenta a reutilização da lógica.


Simplificando com Arrow Function


Explicação:
A Arrow Function permite retorno implícito quando usada em uma única linha, eliminando a necessidade de escrever a palavra return.


Exemplo:



const somar = (valor1, valor2) => valor1 + valor2;

console.log(somar(150, 500));

Resumo:
- Retorno implícito simplifica a sintaxe.
- Ideal para cálculos e funções rápidas.
- Código mais limpo e enxuto.


Formatando Valor Monetário


Explicação:
Para valores monetários, podemos usar a classe Intl.NumberFormat para padronizar o formato de acordo com a região (locale).
new cria um objeto (instância) dessa classe, permitindo acessar suas funcionalidades.


Exemplo:



let preco = 5000;
let desconto = preco * 0.10;
let precoFinal = preco - desconto;

function formatarMoeda(valor) {
    return new Intl.NumberFormat("pt-br", {
        style: "currency",
        currency: "BRL"
    }).format(valor);
}

console.log("Preço Original: " + formatarMoeda(preco));
console.log("Desconto: " + formatarMoeda(desconto));
console.log("Preço final: " + formatarMoeda(precoFinal));

Resumo:
- Intl.NumberFormat ajuda na formatação regional.
- new cria instâncias de classes para usar métodos.
- Útil para exibir valores claros e padronizados.
