1. Fundamentos da Lógica de Programação


  • Sequência: execução dos comandos em ordem.

Explicação:
Significa que as instruções são executadas passo a passo,
seguindo a ordem em que foram escritas no programa.


  • Decisão (condicional): se... então... senão...

Explicação:
Permite ao programa escolher entre diferentes caminhos,
dependendo se uma condição é verdadeira (true) ou falsa (false).


  • Repetição (loop): repetir ações até uma condição ser atendida.

Explicação:
Utilizado quando precisamos executar um conjunto de instruções várias vezes,
até que uma condição de parada seja atingida.


Resumo:
Toda lógica é baseada em três pilares:
executar instruções na ordem certa (sequência),
tomar decisões (condicionais)
e repetir processos sempre que necessário (loops).

2. Variáveis e Constantes


  • Variáveis: guardam valores que podem mudar (let).

Explicação:
São espaços na memória do computador para armazenar dados que podem ser alterados durante a execução do programa.


  • Constantes: valores que não mudam (const).

Explicação:
São valores fixos que, uma vez definidos, não podem ser modificados no decorrer do programa.


  • Tipos comuns: number, string, boolean, undefined.

Explicação:
Cada variável ou constante armazena um tipo de dado:
números (number), textos (string), valores lógicos (boolean) ou ainda sem valor definido (undefined).


let idade = 30;
const nome = "Tiago";
let teste;
teste = "Agora tem valor";
console.log(idade, nome, teste);
// Saída: 30 Tiago Agora tem valor

Resumo:
Variáveis guardam dados que podem mudar,
constantes guardam dados fixos,
e cada valor tem um tipo específico (número, texto, lógico etc).

3. Operadores


Aritméticos

  • + - * / % **
let a = 10, b = 5;
console.log(a + b); // 15
console.log(a % b); // 0

Explicação:
Usados para realizar cálculos matemáticos, como soma, subtração, multiplicação, divisão, resto e potência.


Relacionais

  • == (igualdade de valor)
  • != (diferente de valor)
  • > (maior que)
  • < (menor que)
  • >= (maior ou igual)
  • <= (menor ou igual)
console.log(10 > 5);       // true
console.log(10 == "10");   // true (só compara valor)
console.log(10 != 5);      // true
console.log(10 >= 10);     // true
console.log(5 < 3);        // false

Explicação:
Usados para comparar valores.
O resultado sempre será true (verdadeiro) ou false (falso).


Lógicos

  • && (E)
  • || (OU)
  • ! (negação)
console.log(10 > 5 && 3 < 2); // false

Explicação:
Usados para combinar expressões relacionais.
&& só é verdadeiro se todas forem verdadeiras.
|| é verdadeiro se pelo menos uma for verdadeira.
! inverte o resultado lógico.


Resumo:
Operadores permitem fazer contas,
comparar valores,
e combinar condições para controlar o fluxo do programa.

4. Saída de Dados e Comentários


  • console.log() → imprime informações.

Explicação:
Usado para exibir valores no console do navegador ou do Node.js,
útil para testar o que o programa está fazendo.


  • // → comentário de uma linha.

Explicação:
Permite adicionar observações ou anotações no código,
sem que elas sejam executadas pelo programa.


  • /* ... */ → comentário de várias linhas.

Explicação:
Serve para escrever comentários mais longos,
abrangendo várias linhas, útil para explicar trechos maiores do código.


console.log("Hello World"); // teste
/* Comentário de várias linhas */

Resumo:
Use console.log() para exibir informações e testar seu código,
e comentários para organizar e explicar trechos do programa.

5. Linkando JS com HTML


  • Em linha: código direto no atributo HTML.

Explicação:
Permite colocar pequenos trechos de JavaScript diretamente dentro de elementos HTML,
mas não é recomendado para códigos maiores ou complexos.


  • Interna: dentro da tag <script>.

Explicação:
O JavaScript é escrito dentro do próprio arquivo HTML usando a tag <script>,
bom para testes rápidos ou scripts curtos.


  • Externa: arquivo separado .js (melhor prática).

Explicação:
O código JavaScript é colocado em arquivos separados e vinculados ao HTML,
mantendo o projeto organizado e facilitando a manutenção.


<!-- Interna -->
<script>console.log("JS interno");</script>

<!-- Externa -->
<script src="script.js"></script>

Resumo:
O JavaScript pode ser escrito direto no HTML,
mas o ideal é usar arquivos externos para melhor organização e manutenção do código.

7. Ordem de Cálculo


let a = 100;
let b = 20;
let media = (a + b) / 2; 
console.log(media);
// Saída: 60

Explicação:
O JavaScript segue a ordem matemática padrão: parênteses, multiplicação/divisão e depois soma/subtração.
Usar parênteses ajuda a garantir que as operações sejam executadas na ordem desejada.


Resumo:
Use parênteses para definir a ordem de execução das operações matemáticas e evitar resultados inesperados.

8. Strings


O que é:
Uma string é uma sequência de caracteres (letras, números ou símbolos) usada para representar texto em programação.


Concatenação de Strings

let nome = "Tiago";
let curso = "JS";
console.log("Olá " + nome + ", bem-vindo ao " + curso + "!");
// Saída: Olá Tiago, bem-vindo ao JS!

Template String

let nome = "Tiago";
let curso = "JS";
console.log(`Olá ${nome}, bem-vindo ao ${curso}!`);
// Saída: Olá Tiago, bem-vindo ao JS!

Função principal:
Gerar strings combinadas de texto e variáveis de forma dinâmica e legível.


Resumo:
- Strings representam texto.
- Use + ou template literals ${} para concatenar.
- console.log() exibe o resultado no console.


  • Concatenação = juntar strings com +
  • Template string = montar strings com ${} dentro de crase
  • console.log() = exibir a string
  • Função principal: gerar strings combinadas de texto e variáveis

9. Concatenação e Template String


Concatenação com +

console.log("Olá " + "Mundo");
// Saída: Olá Mundo

Explicação:
Une duas ou mais strings usando o operador +.
Útil para criar mensagens combinando textos fixos e variáveis.


Template String

let nome = "Tiago";
console.log(`Olá ${nome}`);
// Saída: Olá Tiago

Explicação:
Permite inserir variáveis diretamente dentro de strings usando crases ` `
e a sintaxe ${}, tornando o código mais legível e prático.


Concatenação com +=

let frase = "Estudo JS ";
frase += "no Senac.";
console.log(frase);
// Saída: Estudo JS no Senac.

Explicação:
Adiciona texto a uma string existente sem precisar reescrevê-la completamente,
útil para construir frases ou mensagens dinamicamente.


Resumo:
Strings podem ser unidas com + ou de forma mais moderna usando ${} dentro de crases,
e o operador += permite acrescentar conteúdo a uma string existente.

10. Identificando Tipos (typeof)


let numero = 10;
let texto = "Oi";
console.log(typeof numero); // number
console.log(typeof texto);  // string

Explicação:
O operador typeof retorna o tipo de dado armazenado em uma variável.
Pode ser number, string, boolean, undefined, entre outros.


Resumo:
Use typeof para descobrir o tipo de qualquer variável,
ajudando a evitar erros e entender melhor o comportamento do código.
