Introdução aos Objetos

Explicação:
Objetos são estruturas que armazenam dados em pares chave-valor.
Cada chave é um identificador único e cada valor é o conteúdo associado.
Objetos são úteis quando precisamos organizar informações mais complexas que não cabem apenas em uma lista sequencial.

Uma forma fácil de entender objetos é pensar em um armário já aberto.
Cada gaveta tem um nome (chave) e contém itens (valor).
Para acessar um item, basta indicar o nome da gaveta.

Como declarar um Objeto

Explicação:
Um objeto é declarado usando chaves {}.
Cada par chave-valor é separado por vírgula.

let armario = {
            camisa: "azul",
            calca: "preta",
            meias: "vermelhas"
            };

Acessando elementos do Objeto

Explicação:
Para acessar o valor de uma chave, usamos o nome do objeto seguido de ponto e o nome da chave.
Isso é como abrir a gaveta pelo nome.

console.log(armario.camisa); // "azul"
console.log(armario.meias); // "vermelhas"

Modificando elementos

Explicação:
Podemos alterar o valor associado a uma chave existente.
Basta acessar a chave e atribuir um novo valor.

armario.camisa = "vermelha";
console.log(armario.camisa); // "vermelha"

Propriedades e Métodos Importantes

  • Object.keys(obj): retorna um array com todas as chaves do objeto.
  • Object.values(obj): retorna um array com todos os valores do objeto.
  • Object.entries(obj): retorna um array de arrays [chave, valor].

Resumo dos Objetos

Resumo:
Objetos são como armários abertos, onde cada gaveta tem um nome (chave) e contém itens (valor).
Podemos acessar, modificar ou adicionar novos pares chave-valor facilmente.
Métodos como Object.keys(), Object.values() e Object.entries() ajudam a gerenciar e percorrer o conteúdo do objeto.
Objetos permitem organizar dados complexos de forma clara e estruturada.
