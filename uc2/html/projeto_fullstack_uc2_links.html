Trabalhando com Links (ou hiperlink)


A tag < a > (de anchor, âncora) é usada para criar links clicáveis que levam a outras páginas, seções ou arquivos. href é o atributo que define o destino do link, o texto entre as tags é o que aparece clicável.

Link externo


<a href="https://google.com" target="_blank">Google</a>

Google

Link interno (para outra página do mesmo site):


<a href="../index.html" target="_blank">HOME</a>

HOME

Link que abre um aplicativo específico


O método depende do tipo de aplicativo e da plataforma (web, desktop ou mobile), alguns navegadores podem bloquear links de apps por segurança, e em dispositivos móveis, esse tipo de link geralmente funciona melhor. Abaixo estão os exemplos mais comuns:


  • App de e-mail

    
<a href="mailto:emerson.agomes1@senacsp.edu.br">Enviar e-mail</a>
    Enviar e-mail

  • App de telefone

    
<a href="tel:+55156">Ligar agora</a>
    Ligar agora

  • App Whatsapp

    
<a href="https://wa.me/5511940646240">WhatsApp</a>

    WhatsApp


    Abre o app do WhatsApp (ou versão web se estiver no PC).


  • Link para abrir um app* específico (Mobile)

    
<a href="spotify">Abrir Spotify</a>

    Abrir Spotify


    Isso só funciona se o app estiver instalado.
