Como Funciona a Internet


  • Internet como meio de comunicação: Serve para transmitir informações entre pessoas e máquinas.

  • Criação de páginas: Desenvolvedor cria uma página em texto (HTML, CSS, JS).

  • Hospedagem em servidor: Essa página é enviada para um servidor.

  • Domínio: É o nome do caminho que leva até a página (ex: www.site.com).

  • Função do desenvolvedor: Atua entre cliente (negócio) e usuário final, entregando o produto (site).

Linguagem como Comunicação


A linguagem é uma forma de comunicação entre sistemas, pessoas e navegadores.


Tecnologias Web


  • HTML - Linguagem de Marcação que estrutura o conteúdo da página.

  • CSS - Linguagem de estilo que define a aparência visual (cores, fontes, espaçamentos,...) da página. Depende do HTML para aplicar os estilos corretamente.

  • JavaScript - Linguagem de Programação que adiciona interatividade e lógica à página.

Front-End e Back-End


  • Front-End - Tudo o que é renderizado para o cliente (navegador).

  • Back-End - Executado no servidor (dados, lógicas, banco de dados). Alterações que ocorrem internamente e não são visíveis diretamente.

Ferramentas de Desenvolvimento IDE (Editor de Código)


Ambiente para escrever e organizar códigos.


  • VS Code - Ideal para desenvolvimento web.

  • Visual Studio - Focado em aplicações desktop, jogos, mobile.

Boas Práticas


  • Identação - Organização dos blocos de código com recuo para facilitar leitura.

  • Comentários - Explicar o que está sendo feito, especialmente ao aprender algo novo.

Estrutura do HTML


  • Blocos (Block-level Elements) - Ocupam toda a largura da tela (esquerda até a direita). Ex: nav, section, article, aside, main, footer.

  • Linha (Inline Elements) - Ocupam apenas o espaço do conteúdo na mesma linha. Ex: span, a, strong, em.

Sobre as Tags HTML


  • Tag - Elemento pai, define o tipo de conteúdo.

  • Atributos - Informações adicionais de configuração.

  • Valor do Atributo - Define a configuração (ex: type="text").

  • Semânticos - Têm significado claro (ex: alt, title).

  • Não Semânticos - Genéricos (ex: div, span).

Acessibilidade – Atributos Importantes


  • alt - Texto alternativo para imagens.

  • title - Dica ao passar o mouse sobre o elemento.

  • placeholder - Texto temporário em campos de formulário (ex: "Busque aqui").

SEO (Search Engine Optimization)


É o conjunto de técnicas para otimizar um site e melhorar seu posicionamento nos resultados dos buscadores, como o Google. O objetivo é aumentar a visibilidade, tráfego orgânico (gratuito) e atrair mais visitantes qualificados.


Principais práticas de SEO em HTML:


  • title - Título da página (aparece no Google).

  • meta description - Resumo da página nos resultados.

  • alt nas imagens - Descrição para buscadores e acessibilidade.

  • Headings (h1, h2...) - Estrutura clara do conteúdo.

  • URLs amigáveis - Claras e descritivas. (Ex.: https://site.com/produtos/bolsa-ecologica)

  • lang="pt-BR" - Define o idioma da página.

  • Uso de tags semânticas - header, section, article, etc.

Aplicando CSS no HTML


  • Interno (dentro da tag style) - Escreve o CSS no próprio HTML, dentro do head

  • Inline (na própria tag HTML) - Aplica o estilo direto no elemento, com o atributo style

  • Externo (mais recomendado) - Arquivo separado .css, ligado com a tag link no head
