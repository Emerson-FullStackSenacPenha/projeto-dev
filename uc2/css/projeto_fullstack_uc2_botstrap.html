Mídias e Estilização

Tags para mídias

  • Imagem: <img>
  • Áudio: <audio controls>
  • Vídeo: <video controls>

Explicação:
Cada tipo de mídia possui sua própria tag no HTML.
Para áudio e vídeo, o atributo controls adiciona os controles nativos do navegador.
Não podemos deixar as tags soltas: imagens devem estar dentro de <figure> e legendas em <figcaption>.
Para áudio e vídeo, podemos usar <source> dentro da tag principal para fornecer múltiplos formatos.

Exemplo de imagem com legenda

<figure>
  <img src="foto.jpg" alt="Descrição da imagem">
  <figcaption>>Esta é a legenda da imagem.</figcaption>
</figure>

Exemplo de áudio e vídeo com múltiplos formatos

<audio controls>
  <source src="musica.mp3" type="audio/mpeg">
  <source src="musica.ogg" type="audio/ogg">
  Seu navegador não possui rederizador de áudio.
</audio>

<video controls width="300">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Seu navegador não possui rederizador de vídeo.
</video>

Resumo:
Use tags específicas para cada mídia.
Sempre adicione legendas para imagens.
Para áudio e vídeo, forneça múltiplos formatos e mensagem de fallback.

Backgrounds e Transições

Propriedades de background

  • background-image: define a imagem de fundo
  • background-size: cover / contain
  • background-repeat: repeat-x / repeat-y / no-repeat
  • background-position: posição da imagem (center, top, left...)
  • background-attachment: fixed (para efeito parallax)
  • background: linear-gradient(45deg, cor1, cor2)
  • background: radial-gradient(cor1, cor2)

Exemplo de background com gradiente

body {
  background: linear-gradient(45deg, #ff0000, #0000ff);
  background-size: cover;
  background-repeat: no-repeat;
  background-attachment: fixed;
}

Transições e transformações

div {
  transition: transform 0.5s;
}
div:hover {
  transform: scale(1.2);
}

Explicação:
Background pode ser imagem ou gradiente.
Cover ajusta a imagem para cobrir todo o container, contain mantém proporção.
Repeat controla se a imagem se repete horizontal ou verticalmente.
Attachment fixed cria o efeito de parallax.
Transition define o tempo da animação ao interagir com elementos.
Transform permite aumentar, girar ou mover elementos.


Resumo:
Use backgrounds para estilização visual.
Transitions suavizam alterações.
Transform dá movimento e efeitos interativos.

Responsividade e Media Queries

Media Queries

@media (max-width: 600px) {
  body {
    background-color: #f0f0f0;
  }
}

Explicação:
Media queries permitem adaptar o estilo de acordo com o tamanho da tela.
Podemos definir limites máximos ou mínimos de largura, altura ou outras características.
Elas devem ser colocadas no final do CSS para sobrescrever estilos padrões.

Resumo:
Responsividade garante que o site se adapte a diferentes dispositivos.
Media queries são essenciais para controlar estilos em telas pequenas.

Pseudoclasses e Atributos

Propriedades úteis

  • hidden: esconde elementos da tela
  • hover: muda o estilo ao passar o mouse
  • transform: aplica escala, rotação, translação, etc.

Exemplo de hover e hidden

p:hover {
  color: red;
}

div[hidden] {
  display: none;
}

Explicação:
Pseudoclasses como hover permitem interatividade sem JavaScript.
Hidden é um atributo que remove o elemento da renderização, mas mantém no HTML.
Transform modifica visualmente os elementos de forma dinâmica.


Resumo:
Use pseudoclasses para interação visual.
Hidden para esconder elementos sem apagar do código.
Transform para animações e efeitos.
