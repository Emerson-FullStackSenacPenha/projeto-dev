Utilizando FlexBox


Flexbox permite criar layouts responsivos, organizando os elementos de forma flexível, em linha ou coluna, substituindo o uso tradicional de float, inline-block, etc.


Para ativa-lo usamos "display: flex;" no elemento pai.

Direção

  • flex-direction: row → Elementos em linha (padrão).
  • flex-direction: column → Elementos em coluna.

Alinhamentos

  • justify-content: Alinha os itens no eixo principal (horizontal por padrão).
  • align-items: Alinha os itens no eixo cruzado (vertical por padrão).

Manipulação de comportamento

  • flex-grow: Define qual item deve crescer mais.
  • flex-shrink: Define qual item pode encolher mais.
  • flex-basis: Tamanho base inicial do item (antes de crescer/encolher).

Comportamento

  • flex-wrap: Permite que os itens quebrem linha quando não couberem.
  • gap: Espaçamento entre os itens.
