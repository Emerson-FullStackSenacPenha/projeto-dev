Bordas e margens


Padrão para começar o arquivo do css, resetando bordas e margens


* {

margin: 0;

padding: 0;

}


Para visualizar melhor, usar border ou background-color. Se o elemento pai não tem tamanho definido, os filhos definem a altura/largura.

Tamanhos e Unidades de Medida


width = Largura

height = Altura

Unidades


  • px: pixels (unidade fixa)
  • %: porcentagem em relação ao elemento pai (menos comum)
  • vw: largura da tela (viewport)
  • vh: altura da tela (viewport)
  • em: relativo ao tamanho da fonte do elemento pai
  • rem: relativo ao tamanho da fonte raiz (root)

Espaçamentos


  • Padding: Espaço interno do elemento.
  • Margin: Espaço externo ao redor do elemento.

Ordem dos valores


  • 1 valor: todos os lados (1px)
  • 2 valores: cima/baixo e esquerda/direita (1px 1px)
  • 4 valores: cima - direita - baixo - esquerda (1px 1px 1px 1px)
