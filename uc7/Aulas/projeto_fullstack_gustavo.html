Regras para o projeto Integrador


  1. Esse projeto é feito fora de sala
    O desenvolvimento do projeto não será feito durante as aulas; o grupo deve se organizar para trabalhar em horários extra-classe.

  2. Quem falta não decide nada
    Quem não comparecer aos encontros ou reuniões do grupo perde o direito de participar das decisões.

  3. Atrasos, qualidade e principalmente finalização
    O grupo deve ter atenção aos prazos, manter a qualidade do trabalho e, principalmente, garantir que o projeto seja concluído.

  4. Se não está documentado, não existe!
    Tudo que for feito precisa estar registrado (documentação técnica, atas, etc.). Se não houver registro, será considerado como se não tivesse sido feito.
