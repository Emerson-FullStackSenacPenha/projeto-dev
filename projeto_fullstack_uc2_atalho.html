Atalhos úteis para HTML no VS Code

No Visual Studio Code, existem diversos atalhos que facilitam muito a vida na hora de escrever código HTML. Aqui estão alguns dos mais usados:

  • Duplicar linha ou seleção: Shift + Alt + ↓ Duplica a linha atual ou a seleção para baixo.
  • Selecionar múltiplas palavras iguais: Ctrl + D Seleciona a próxima ocorrência da palavra ou trecho selecionado. Pressione várias vezes para selecionar todas as ocorrências.
  • Gerar apenas texto Lorem rápido: lorem + Tab Cria um pequeno parágrafo de texto fictício.
  • Criar múltiplos parágrafos com texto fictício (lorem): (p>lorem)*8 + Tab Gera 8 parágrafos com texto de preenchimento (lorem ipsum) automaticamente.
  • Tag dentro de outra tag: p > b > i + Tab Cria uma estrutura de tags aninhadas
